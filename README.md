# Marketing_and_Sales_Analysis
Data analysis and linear regression project exploring the relationship between advertising spend and sales using Python

# Project Overview
This project analyzes the relationship between advertising expenditure and product sales using Python. The objective is to understand how spending on different advertising channels influences sales and to build a predictive model that estimates sales based on advertising investments.
The project includes data cleaning, exploratory data analysis (EDA), visualization, correlation analysis, regression modeling, and return on investment (ROI) calculations.

# Objectives
Clean and prepare the advertising dataset for analysis.
Perform exploratory data analysis to identify trends and patterns.
Examine the relationship between advertising channels and sales.
Identify the advertising channel with the strongest impact on sales.
Build a linear regression model to predict sales.
Calculate the return on investment (ROI) for advertising channels.
